# AI Product Development Engineer (Intern) — Take-Home Assignment

**InsideIIM × Altuni AI Labs**  
**Role:** Remote · paid · [1-year], with conversion to a full-time role + PPO for strong performers.

## The Task
Build an **AI Investment Research Agent**:
* It takes a company name, does its research, and decides whether to invest or pass — with the reasoning behind its decision.
* How you build it (what it researches, how it works under the hood, how it shows its results) is entirely up to you.
* Focus is on **what you build** and **how you build it**.

## Tech Stack (Production Stack — Mandatory to build with)
* **Frontend:** React or Next.js
* **Backend:** Node.js or Next.js
* **AI Framework:** LangChain.js or LangGraph.js
* **LLM / APIs / Tools:** Any LLM provider and libraries of your choice.
* **Deployment:** Deploying online (e.g. on Vercel) carries **bonus points**.

## Deliverables
A zip file containing the code and a `README.md` with:
1. **Overview** — What it does.
2. **How to run it** — Setup, run steps, environment keys needed.
3. **How it works** — Approach and architecture.
4. **Key decisions & trade-offs** — Choices, rationale, and what was left out.
5. **Example runs** — Agent's output on a few companies of your choice.
6. **Future improvements** — What you would improve with more time.
7. **BONUS:** Chat logs / transcript of your sessions with the AI/LLM while building this project (shows your thought process).

## Ground Rules
* Strictly solo assignment. Code sharing will be detected.
* You must be able to explain everything you submit.
* If anything is ambiguous, make your own call and document it in the README.
